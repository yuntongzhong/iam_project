Authority 映射说明

这部分是给答辩讲解准备的：Spring Security 最终并不直接判断“角色对象”或“权限对象”，而是判断运行时 Authority 字符串。

`ROLE_ADMIN`

表示角色 Authority。由角色编码 `ADMIN` 映射而来，供 `hasRole('ADMIN')` 这类表达式使用。

`PERM_USER_WRITE`

表示权限 Authority。由权限编码 `USER_WRITE` 映射而来，供 `hasAuthority('PERM_USER_WRITE')` 使用。

角色与权限的关系

角色是业务身份包，权限是最小动作单元。Spring 在运行时把两者都铺平成 Authority，再参与访问控制判断。
